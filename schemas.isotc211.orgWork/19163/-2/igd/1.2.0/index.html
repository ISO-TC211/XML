Image and Gridded Data (IGD) Version 1.2.0

Description

Image and Gridded Data (IGD) is an XML Schema implementation of abstract classes for ISO 19163, Geographic Information - Content components and encoding rules for imagery and gridded Data. It includes elements for defining geometry and functions for coverages. The XML schema was encoded using the encoding rules defined in ISO 19118, ISO 19139.

XML Namespace for Image and Gridded Data (IGD) Version: 1.2

The namespace URI for Image and Gridded Data (IGD) Version 1.2.0 is https://schemas.isotc211.org/19163/-2/igd/1.2.

XML Schema for Image and Gridded Data (IGD) Version 1.2.0

igd.xsd is the XML Schema document to be referenced by XML documents containing XML elements in the igd namespace or by XML Schema documents importing the igd 1.2 namespace. This XML schema includes (indirectly) all the implemented concepts of the igd namespace, but it does not contain the declaration of any types.
