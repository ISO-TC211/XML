Geographic Markup Wrappers (GWM) Version: 1.0

Description

GWM 1.0 is an XML Schema implementation derived from ISO ISO 19115-1, Geographic Information - Metadata - Part 1: Fundamentals, Clause TBD. It includes wrappers for gml dependencies. The XML schema was encoded using the rules described in ISO/TS 19139:2007, Clause 8 and implementation approach from ISO/TS19115-3, Clause 8.

XML Namespace for gwm 1.0

The namespace URI for gwm 1.0 is http://standards.iso.org/19115/-3/gwm/1.0.

XML Schema for gwm 1.0

gwm.xsd is the XML Schema document to be referenced by XML documents containing XML elements in the gwm 1.0 namespace or by XML Schema documents importing the gwm 1.0 namespace. This XML schema includes (indirectly) all the implemented concepts of the gwm namespace, but it does not contain the declaration of any types.

NOTE: The XML Schema for gwm 1.0 are available here as part of a zip archive including all the XML Schema Implementations defined in ISO/TS 19115-3.

No Related XML Namespaces for gwm 1.0
