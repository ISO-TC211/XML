Metadata for Portrayal Catalog (MPC) Version: 1.0

Description

MPC 1.0 is an XML Schema implementation derived from ISO ISO 19115-1, Geographic Information - Metadata - Part 1: Fundamentals, Clause 6.5.10. It includes elements for referencing portrayal catalogs. The XML schema was encoded using the rules described in ISO/TS 19139:2007, Clause 8 and implementation approach from ISO/TS19115-3, Clause 8.

XML Namespace for mpc 1.0

The namespace URI for mpc 1.0 is http://standards.iso.org/19115/-3/mpc/1.0.

XML Schema for mpc 1.0

mpc.xsd is the XML Schema document to be referenced by XML documents containing XML elements in the mpc 1.0 namespace or by XML Schema documents importing the mpc 1.0 namespace. This XML schema includes (indirectly) all the implemented concepts of the mpc namespace, but it does not contain the declaration of any types.

NOTE: The XML Schema for mpc 1.0 are available here as part of a zip archive including all the XML Schema Implementations defined in ISO/TS 19115-3.

Related XML Schema for mpc 1.0

portrayalCatalogue.xsd implements the UML conceptual schema defined in ISO 19115-1, Geographic Information - Metadata - Part 1: Fundamentals, Clause 6.5.10. It was created using the encoding rules defined in ISO 19118, ISO 19139, and the implementation approach described in ISO 19115-3 and contains the following classes (codeLists are bold): MD_PortrayalCatalogueReference
