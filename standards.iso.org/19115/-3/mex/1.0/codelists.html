MD_DatatypeCode ( MD_DatatypeCode ):

Description: datatype of element or entity
Entry Definition
MD_DatatypeCode_class descriptor of a set of objects that share the same attributes, operations, methods, relationships, and behaviour
MD_DatatypeCode_codelist flexible enumeration useful for expressing a long list of values, can be extended
MD_DatatypeCode_enumeration data type whose instances form a list of named literal values, not extendable
MD_DatatypeCode_codelistElement permissible value for a codelist or enumeration
MD_DatatypeCode_abstractClass class that cannot be directly instantiated
MD_DatatypeCode_aggregateClass class that is composed of classes it is connected to by an aggregate relationship
MD_DatatypeCode_specifiedClass subclass that may be substituted for its superclass
MD_DatatypeCode_datatypeClass class with few or no operations whose primary purpose is to hold the abstract state of another class for transmittal, storage, encoding or persistent storage
MD_DatatypeCode_interfaceClass named set of operations that characterize the behaviour of an element
MD_DatatypeCode_unionClass class describing a selection of one of the specified types
MD_DatatypeCode_metaClass class whose instances are classes
MD_DatatypeCode_typeClass class used for specification of a domain of instances (objects), together with the operations applicable to the objects. A type may have attributes and associations
MD_DatatypeCode_characterString textual information
MD_DatatypeCode_integer numerical field
MD_DatatypeCode_association semantic relationship between two classes that involves connections among their instances
