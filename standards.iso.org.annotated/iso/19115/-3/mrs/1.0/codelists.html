MD_ReferenceSystemTypeCode ( MD_ReferenceSystemTypeCode ):

Description: defines type of reference system used
Entry Definition
MD_ReferenceSystemTypeCode_compoundEngineeringParametric compound statio-parametric coordinate reference system containing an engineering coordinate reference system and a parametric reference system. e.g. [local] x, y, pressure
MD_ReferenceSystemTypeCode_compoundEngineeringParametricTemporal compound statio-parametric-temporal coordinate reference system containing an engineering, a parametric, and a temporal coordinate reference system. e.g. [local] x, y, pressure, time
MD_ReferenceSystemTypeCode_compoundEngineeringTemporal compound spatio-temporal coordinate reference system containing an engineering and a temporal coordinate reference system. e.g. [local] x, y, time
MD_ReferenceSystemTypeCode_compoundEngineeringVertical compound spatial reference system containing a horizontal engineering coordinate reference system and a vertical coordinate reference system. e.g. [local] x, y, height
MD_ReferenceSystemTypeCode_compoundEngineeringVerticalTemporal compound spatio-temporal coordinate reference system containing an engineering, a vertical, and a temporal coordinate reference system. e.g. [local] x, y, height, time
MD_ReferenceSystemTypeCode_compoundGeodeticParametric compound statio-parametric coordinate reference system containing a 2 dimensional geographic horizontal coordinate reference system and a parametric reference system. e.g. latitude, longitude, pressure
MD_ReferenceSystemTypeCode_compoundGeodeticParametricTemporal compound statio-parametric-temporal coordinate reference system containing a 2 dimensional geographic horizontal, a parametric and a temporal coordinate reference system. e.g. latitude, longitude, pressure, time
MD_ReferenceSystemTypeCode_compoundGeographic2DTemporal compound spatio-temporal coordinate reference system containing a 2 dimensional geographic horizontal coordinate reference system and a temporal reference system. e.g. latitude, longitude, time
MD_ReferenceSystemTypeCode_compoundGeographic2DVertical compound coordinate reference system in which one constituent coordinate reference system is a horizontal geodetic coordinate reference system and one is a vertical coordinate reference system. e.g. latitude, longitude, [gravity-related] height or depth
MD_ReferenceSystemTypeCode_compoundGeographicVerticalTemporal compound spatio-temporal coordinate reference system containing a 2 dimensional geographic horizontal, a vertical, and a temporal coordinate reference system. e.g. latitude, longitude, height, time
MD_ReferenceSystemTypeCode_compoundGeographic3DTemporal compound spatio-temporal coordinate reference system containing a 3 dimensional geographic and a temporal coordinate reference system. e.g. latitude, longitude, ellipsoidal height, time
MD_ReferenceSystemTypeCode_compoundProjected2DParametric compound statio-parametric coordinate reference system containing a projected horizontal coordinate reference system and a parametric reference system. e.g. easting, northing, density
MD_ReferenceSystemTypeCode_compoundProjected2DParametricTemporal compound statio-parametric-temporal coordinate reference system containing a projected horizontal, a parametric, and a temporal coordinate reference system. e.g. easting, northing, density, time
MD_ReferenceSystemTypeCode_compoundProjectedTemporal compound spatio-temporal coordinate reference system containing a projected horizontal and a temporal coordinate reference system. e.g. easting, northing, time
MD_ReferenceSystemTypeCode_compoundProjectedVertical compound spatial reference system containing a horizontal projected coordinate reference system and a vertical coordinate reference system. e.g. easting, northing, [gravity-related] height or depth
MD_ReferenceSystemTypeCode_compoundProjectedVerticalTemporal compound spatio-temporal coordinate reference system containing a projected horizontal, a vertical, and a temporal coordinate reference system. e.g. easting, northing, height, time
MD_ReferenceSystemTypeCode_engineering coordinate reference system based on an engineering datum (datum describing the relationship of a coordinate system to a local reference). e.g. [local] x,y
MD_ReferenceSystemTypeCode_engineeringDesign engineering coordinate reference system in which the base representation of a moving object is specified. e.g. [local] x,y
MD_ReferenceSystemTypeCode_engineeringImage coordinate reference system based on an image datum (engineering datum which defines the relationship of a coordinate system to an image). e.g. row, column
MD_ReferenceSystemTypeCode_geodeticGeocentric geodetic CRS having a Cartesian 3D coordinate system. e.g. [geocentric] X,Y,Z
MD_ReferenceSystemTypeCode_geodeticGeographic2D geodetic CRS having an ellipsoidal 2D coordinate system. e.g. latitude, longitude
MD_ReferenceSystemTypeCode_geodeticGeographic3D geodetic CRS having an ellipsoidal 3D coordinate system. e.g. latitude, longitude, ellipsoidal height
MD_ReferenceSystemTypeCode_geographicIdentifier spatial reference in the form of a label or code that identifies a location. e.g. post code
MD_ReferenceSystemTypeCode_linear set of Linear Referencing Methods and the policies, records and procedures for implementing them. reference system that identifies a location by reference to a segment of a linear geographic feature and distance along that segment from a given point. e.g. x km along road
MD_ReferenceSystemTypeCode_parametric coordinate reference system based on a parametric datum (datum describing the relationship of a parametric coordinate system to an object). e.g. pressure
MD_ReferenceSystemTypeCode_projected coordinate reference system derived from a two-dimensional geodetic coordinate reference system by applying a map projection. e.g. easting, northing
MD_ReferenceSystemTypeCode_temporal reference system against which time is measured. e.g. time
MD_ReferenceSystemTypeCode_vertical one-dimensional coordinate reference system based on a vertical datum (datum describing the relation of gravity-related heights or depths to the Earth). e.g. [gravity-related] height or depth
