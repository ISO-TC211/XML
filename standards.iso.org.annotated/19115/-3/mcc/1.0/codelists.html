ISO 19115-3 Codelist Report

This report describes the ISO 19115-3 codelists and their values.

Please contact Ted Habermann if you have questions or suggestions.

Codelists:

MD_ProgressCode :

Description: status of the resource
Entry Definition
MD_ProgressCode_completed has been completed
MD_ProgressCode_historicalArchive stored in an offline storage facility
MD_ProgressCode_obsolete no longer relevant
MD_ProgressCode_onGoing continually being updated
MD_ProgressCode_planned fixed date has been established upon or by which the resource will be created or updated
MD_ProgressCode_required needs to be generated or updated
MD_ProgressCode_underDevelopment currently in the process of being created
MD_ProgressCode_final progress concluded and no changes will be accepted
MD_ProgressCode_pending committed to, but not yet addressed
MD_ProgressCode_retired item is no longer recommended for use. It has not been superseded by another item
MD_ProgressCode_superseded replaced by new
MD_ProgressCode_tentative provisional changes likely before resource becomes final or complete
MD_ProgressCode_valid acceptable under specific conditions
MD_ProgressCode_accepted agreed to by sponsor
MD_ProgressCode_notAccepted rejected by sponsor
MD_ProgressCode_withdrawn removed from consideration
MD_ProgressCode_proposed suggested that development needs to be undertaken
MD_ProgressCode_deprecated resource superseded and will become obsolete, use only for historical purposes
top

MD_ScopeCode :

Description: class of information to which the referencing entity applies
Entry Definition
MD_ScopeCode_attribute information applies to the attribute value
MD_ScopeCode_attributeType information applies to the characteristic of a feature
MD_ScopeCode_collectionHardware information applies to the collection hardware class
MD_ScopeCode_collectionSession information applies to the collection session
MD_ScopeCode_dataset information applies to the dataset
MD_ScopeCode_series information applies to the series
MD_ScopeCode_nonGeographicDataset information applies to non-geographic data
MD_ScopeCode_dimensionGroup information applies to a dimension group
MD_ScopeCode_feature information applies to a feature
MD_ScopeCode_featureType information applies to a feature type
MD_ScopeCode_propertyType information applies to a property type
MD_ScopeCode_fieldSession information applies to a field session
MD_ScopeCode_software information applies to a computer program or routine
MD_ScopeCode_service information applies to a capability which a service provider entity makes available to a service user entity through a set of interfaces that define a behaviour, such as a use case
MD_ScopeCode_model information applies to a copy or imitation of an existing or hypothetical object
MD_ScopeCode_tile information applies to a tile, a spatial subset of geographic data
MD_ScopeCode_metadata information applies to metadata
MD_ScopeCode_initiative information applies to an initiative
MD_ScopeCode_sample information applies to a sample
MD_ScopeCode_document information applies to a document
MD_ScopeCode_repository information applies to a repository
MD_ScopeCode_aggregate information applies to an aggregate resource
MD_ScopeCode_product metadata describing an ISO 19131 data product specification
MD_ScopeCode_collection information applies to an unstructured set
MD_ScopeCode_coverage information applies to a coverage
MD_ScopeCode_application information resource hosted on a specific set of hardware and accessible over a network

MD_SpatialRepresentationTypeCode :

Description: method used to represent geographic information in the resource
Entry Definition
MD_SpatialRepresentationTypeCode_vector vector data are used to represent geographic data
MD_SpatialRepresentationTypeCode_grid grid data are used to represent geographic data
MD_SpatialRepresentationTypeCode_textTable textual or tabular data are used to represent geographic data
MD_SpatialRepresentationTypeCode_tin triangulated irregular network
MD_SpatialRepresentationTypeCode_stereoModel three-dimensional view formed by the intersecting homologous rays of an overlapping pair of images
MD_SpatialRepresentationTypeCode_video scene from a video recording
