Metadata for Data and Services (MDS) Version: 1.0

Description

MDS 1.0 is an XML Schema implementation derived from ISO ISO 19115-1, Geographic Information - Metadata - Part 1: Fundamentals, Clause 6.5.2. It includes this namespace contains all classes needed for a complete metadata record for any resource. It is used for metadata records that do bnot include geospatial common extensions (gex), extenbsions to the schema (mex), data series (DS_*) elements (mda) or metadata for data transfer (mdt). The XML schema was encoded using the rules described in ISO/TS 19139:2007, Clause 8 and ISO/TS19115-3, Clause 8.

XML Namespace for mds 1.0

The namespace URI for mds 1.0 is http://www.isotc211.org/namespace/mds/1.0.

XML Schema for mds 1.0

mds.xsd is the XML Schema document to be referenced by XML documents containing XML elements in the mds 1.0 namespace or by XML Schema documents importing the mds 1.0 namespace. This XML schema includes (indirectly) all the implemented concepts of the mds namespace, but it does not contain the declaration of any types.

NOTE: The XML Schema for mds 1.0 are available here as part of a zip archive including all the XML Schema Implementations defined in ISO/TS 19115-3.

Schematron Validation Rules for mds 1.0
