Metadata for Resource Content (MRC) Version: 1.0

Description

MRC 1.0 is an XML Schema implementation derived from ISO ISO 19115-1, Geographic Information - Metadata - Part 1: Fundamentals, Clause 6.5.9. It includes elements for describing resource structure and content. The XML schema was encoded using the rules described in ISO/TS 19139:2007, Clause 8 and ISO/TS19115-3, Clause 8.

XML Namespace for mrc 1.0

The namespace URI for mrc 1.0 is http://www.isotc211.org/namespace/mrc/1.0.

XML Schema for mrc 1.0

mrc.xsd is the XML Schema document to be referenced by XML documents containing XML elements in the mrc 1.0 namespace or by XML Schema documents importing the mrc 1.0 namespace. This XML schema includes (indirectly) all the implemented concepts of the mrc namespace, but it does not contain the declaration of any types.

NOTE: The XML Schema for mrc 1.0 are available here as part of a zip archive including all the XML Schema Implementations defined in ISO/TS 19115-3.

Related XML Schema for mrc 1.0

content.xsd implements the UML conceptual schema defined in ISO 19115-1, 6.5.9. It was created using the encoding rules defined in ISO 19118, ISO 19139, and ISO 19115-3 and contains the following classes: MD_AttributeGroup, MD_Band, AbstractMD_ContentInformation, MD_CoverageContentTypeCode, MD_CoverageDescription, MD_FeatureCatalogue, MD_FeatureCatalogueDescription, MD_FeatureTypeInfo, MD_ImageDescription, MD_ImagingConditionCode, MD_RangeDimension, and MD_SampleDimension

contentInformationImagery.xsd implements the UML conceptual schema defined in ISO 19115-1, 6.5.9. It was created using the encoding rules defined in ISO 19118, ISO 19139, and ISO 19115-3 and contains the following classes: MI_Band, MI_BandDefinition, MI_CoverageDescription, MI_ImageDescription, MI_PolarisationOrientationCode, MI_RangeElementDescription, and MI_TransferFunctionTypeCode
