Metadata for Maintenance Information (MMI) Version: 1.3

Description

MMI 1.3 is an XML Schema implementation derived from ISO ISO 19115-1, Geographic Information - Metadata - Part 1: Fundamentals, Clause 6.5.6. It includes elements describing resource and metadata maintenance. The XML schema was encoded using the rules described in ISO/TS 19139:2007.

XML Namespace for mmi 1.3

The namespace URI for mmi 1.3 is https://schemas.isotc211.org/19115/-1/mmi/1.3.

XML Schema for mmi 1.3

mmi.xsd is the XML Schema document to be referenced by XML documents containing XML elements in the mmi 1.3 namespace or by XML Schema documents importing the mmi 1.3 namespace. This XML schema includes (indirectly) all the implemented concepts of the mmi namespace, but it does not contain the declaration of any types.

Related XML Schema for mmi 1.3

maintenance.xsd implements the UML conceptual schema defined in ISO 19115-1, Geographic Information - Metadata - Part 1: Fundamentals, Clause 6.5.6. It was created using the encoding rules defined in ISO 19118, ISO 19139.

https://schemas.isotc211.org/19115/-1/mmi/1.3.0/maintenance.xsd contains the folowing class:
  • MD_MaintenanceInformation
https://schemas.isotc211.org/19115/-1/mmi/1.3.0/maintenance.xsd contains references to the folowing codelist:
  • MD_MaintenanceFrequencyCode
