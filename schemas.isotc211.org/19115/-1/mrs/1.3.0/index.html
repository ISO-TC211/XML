Metadata for Reference System (MRS) Version: 1.3

Description

MRS 1.3 is an XML Schema implementation derived from ISO ISO 19115-1, Geographic Information - Metadata - Part 1: Fundamentals, Clause 6.5.8. It includes elements for describing Reference Systems. The XML schema was encoded using the rules described in ISO/TS 19139:2007.

XML Namespace for mrs 1.3

The namespace URI for mrs 1.3 is https://schemas.isotc211.org/19115/-1/mrs/1.3.

XML Schema for mrs 1.3

mrs.xsd is the XML Schema document to be referenced by XML documents containing XML elements in the mrs 1.3 namespace or by XML Schema documents importing the mrs 1.3 namespace. This XML schema includes (indirectly) all the implemented concepts of the mrs namespace, but it does not contain the declaration of any types.
