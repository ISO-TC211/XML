Metadata for Spatial Representation (MSR) Version: 1.3

Description

MSR 1.3 is an XML Schema implementation derived from ISO ISO 19115-1, Geographic Information - Metadata - Part 1: Fundamentals AMENDMENT 1, Clause 6.5.7. It includes elements for describing spatial representations of resources. The XML schema was encoded using the rules described in ISO/TS 19139:2007.

XML Namespace for msr 1.3

The namespace URI for msr 1.3 is https://schemas.isotc211.org/19115/-1/msr/1.3.

XML Schema for msr 1.3

msr.xsd is the XML Schema document to be referenced by XML documents containing XML elements in the msr 1.3 namespace or by XML Schema documents importing the msr 1.3 namespace. This XML schema includes (indirectly) all the implemented concepts of the msr namespace, but it does not contain the declaration of any types.

Related XML Schema for msr 1.3

spatialRepresentation.xsd implements the UML conceptual schema defined in ISO 19115-1, Geographic Information - Metadata - Part 1: Fundamentals AMENDMENT 1, Clause 6.5.7. It was created using the encoding rules defined in ISO 19118, ISO 19139.

https://schemas.isotc211.org/19115/-1/msr/1.3.0/spatialRepresentation.xsd contains the folowing classes:
  • AbstractMD_SpatialRepresentation
  • MD_GridSpatialRepresentation
  • MD_Georectified
  • MD_Georeferenceable
  • MD_VectorSpatialRepresentation
  • MD_Dimension
  • MD_GeometricObjects
https://schemas.isotc211.org/19115/-1/msr/1.3.0/spatialRepresentation.xsd contains references to the folowing codelists:
  • MD_DimensionNameTypeCode
  • MD_CellGeometryCode
  • MD_GeometricObjectTypeCode
  • MD_PixelOrientationCode
  • MD_TopologyLevelCode

spatialRepresentationImagery.xsd implements the UML conceptual schema defined in ISO 19115-2, Geographic Information - Metadata - Part 2: Extensions for acquisition and processing, Clause 6.5.7. It was created using the encoding rules defined in ISO 19118, ISO 19139, and the implementation approach described in ISO 19115-3 and contains the following classes (codeLists are bold): MI_GCP, MI_GCPCollection, AbstractMI_GeolocationInformation, MI_Georectified, and MI_Georeferenceable
