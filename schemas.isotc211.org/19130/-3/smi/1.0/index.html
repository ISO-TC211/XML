XML Namespace for smi 1.0

The namespace URI for smi 1.0 is https://schemas.isotc211.org/19130/-3/smi/1.0.

XML Schema for smi 1.0

smi.xsd is the XML Schema document to be referenced by XML documents containing XML elements in the smi 1.0 namespace or by XML Schema documents importing the smi 1.0 namespace. This XML schema includes (indirectly) all the implemented concepts of the smi namespace, but it does not contain the declaration of any types.
