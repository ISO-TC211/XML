SAR, INSAR, lidar and sonar Version: 1.1

Description

SMI 1.1 is an XML Schema implementation derived from ISO ISO 19130-2, Geographic Information - Imagery Sensor Models for Geopositioning - Part 2: SAR, INSAR, lidar and sonar. The XML schema was encoded using the rules described in ISO/TS 19139:2007.

XML Namespace for smi 1.1

The namespace URI for smi 1.1 is https://schemas.isotc211.org/19130/-1/smi/1.1.

XML Schema for smi 1.1

https://schemas.isotc211.org/19130/-3/smi/1.1.0/smi.xsd is the XML Schema document to be referenced by XML documents containing XML elements in the smi 1.1 namespace or by XML Schema documents importing the smi 1.1 namespace. This XML schema includes (indirectly) all the implemented concepts of the smi namespace, but it does not contain the declaration of any types.
