CATalogue (CAT) Version: 1.2.0

Description

CAT 1.2.0 is an XML Schema implementation derived from ISO 19139:2007, Geographic Information - Metadata - XML Schema Implementation. It includes elements for codelist catalogues, and example catalogues. The XML schema was encoded using the rules described in ISO/TS 19139:2007.

XML Namespace for cat 1.2

The namespace URI for cat 1.2 is https://schemas.isotc211.org/19139/-/cat/1.2.

XML Schema for cat 1.2

cat.xsd is the XML Schema document to be referenced by XML documents containing XML elements in the cat 1.2 namespace or by XML Schema documents importing the cat 1.2 namespace. This XML schema includes (indirectly) all the implemented concepts of the cat namespace, but it does not contain the declaration of any types.
