Metadata for Data Quality (MDQ) Version 1.2.0

Description

Metadata for Data Quality (MDQ) is an XML Schema implementation of abstract classes for ISO 19157, Geographic Information - Data Quality. It includes elements for defining geometry and functions for coverages. The XML schema was encoded using the encoding rules defined in ISO 19118, ISO 19139.

XML Namespace for Metadata for Data Quality (MDQ) Version: 1.2

The namespace URI for Data Quality [abstract classes] (MDQ) Version 1.2.0 is https://schemas.isotc211.org/19157/-/mdq/1.2.

XML Schema for Metadata for Data Quality (MDQ) Version 1.2.0

mdq.xsd is the XML Schema document to be referenced by XML documents containing XML elements in the mdq namespace or by XML Schema documents importing the mdq 1.2 namespace. This XML schema includes (indirectly) all the implemented concepts of the mdq namespace, but it does not contain the declaration of any types.
